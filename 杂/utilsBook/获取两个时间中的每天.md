# 2--获取两个时间中的每天

场景：一个根据时间筛选的图表，后端只返回有数据的那些天，没数据的时候不返回，此时需要前端补上

```javascript
function getDaysArr(timeStart, timeEnd) {
  let dayArr = [];
  for (
    let d = Date.parse(timeStart);
    d < Date.parse(timeEnd);
    d += 24 * 60 * 60 * 1000
  ) {
    dayArr.push(new Date(d + 8 * 60 * 60 * 1000).toJSON().split("T")[0]);
  }
  return dayArr;
}
getDaysArr("2019/02/01", "2019-02-28"); //["2019-02-01", "2019-02-02", "2019-02-03", "2019-02-04", "2019-02-05", "2019-02-06", "2019-02-07", "2019-02-08", "2019-02-09", "2019-02-10", "2019-02-11", "2019-02-12", "2019-02-13", "2019-02-14", "2019-02-15", "2019-02-16", "2019-02-17", "2019-02-18", "2019-02-19", "2019-02-20", "2019-02-21", "2019-02-22", "2019-02-23", "2019-02-24", "2019-02-25", "2019-02-26", "2019-02-27", "2019-02-28"]
```
